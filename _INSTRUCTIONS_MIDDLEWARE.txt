INSTRUCTIONS - Middleware a enregistrer dans bootstrap/app.php

Ajoutez ces middlewares dans votre bootstrap/app.php:

->withMiddleware(function (Middleware $middleware) {
    $middleware->alias([
        'admin' => \App\Http\Middleware\AdminMiddleware::class,
        'kyc' => \App\Http\Middleware\KycMiddleware::class,
        'transaction.access' => \App\Http\Middleware\EnsureTransactionAccess::class,
        'log.activity' => \App\Http\Middleware\LogActivity::class,
    ]);
})

Note: Le middleware 'admin' et 'kyc' sont deja references dans les routes web.php.
